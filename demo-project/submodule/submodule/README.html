Bash Demo Project for Cider-CI

This is a demonstration project for Cider-CI. Every task is based on a simple bash script.

Almost all features of the Cider-CI dot-file .cider-ci.yml and the Cider-CI specification .cider-ci/ are covered by the given examples.
