The Context

Container for tasks and related properties.

Key/Property Presence Value
script_defaults optional Map - Every [script][] will inherit these properties. Examples: [exclusive-executor-resource], [script-dependencies]
subcontexts optional Map - An inheritance mechanism, see the also the Advanced Topics page. Example: [contexts]
task_defaults optional Map - Every task in the scope of this context will inherit these properties. Example: [contexts]
generate_tasks optional Map - See [Generating Tasks] in Advanced Topics. Example: [generate-tasks]
tasks optional1 Map of Maps - See the task.

  1. The presence of the tasks property and its content is optional from the configuration file perspective. However, a job without tasks makes no sense in Cider-CI and will result with the state failed. ↩
